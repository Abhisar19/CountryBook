About CountryBook

CountryBook provides detailed information about various countries around the world. Since the project's main motive is to showcase front-end capabilities and not backend functionalities, the chosen project has been kept simple in terms of functionalities. CountryBook aims to encapsulate information sourced from websites such as Britannica, Wikipedia, among others, in a more dynamic and user-friendly manner. Initially, I will focus on achieving core functionality and developing more modules to help manage countries' information. Other extensions will be decided once the core functionalities are completed.

CountryBook is expected to have the following features:

  • An alphabetically arranged list of countries with their individual pages.
  • Proper routing ensuring better user experience.
  • Hierarchical page display.
  • A special section for summary on each page to ensure quick dissemination of information.
  • Use of high quality open source icons and images for better user experience.
  • A more dynamic and responsive user interface with the help of navigation bars, buttons, breadcrumbs, etc.
